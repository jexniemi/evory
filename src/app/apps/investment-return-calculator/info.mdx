## What is ROI?

ROI (Return on Investment) or return on invested capital tells how much the investment has yielded relative to the original capital. It is one of the most used key figures for comparing the profitability of investments and business.

## Formulas

**Total return:**
ROI = (End value − Initial investment) / Initial investment × 100 %

**Annualized annual return (CAGR):**
CAGR = (End value / Initial investment)^(1/years) − 1

CAGR is useful for comparing investments of different lengths, as it converts the total return to an annual return percentage.

## When to use ROI and when CAGR?

- **ROI** is suitable for assessing the total return of a single investment
- **CAGR** is suitable for comparing investments of different lengths, as it takes into account the effect of time

For example, 100% return in 10 years sounds good, but CAGR is only 7.2%/y. On the other hand, 20% return in 2 years corresponds to 9.5% annual return.
