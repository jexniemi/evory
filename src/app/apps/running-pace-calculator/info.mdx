## Speed vs. Pace

**Speed** (km/h) tells how many kilometers you run per hour. **Pace** (min/km) tells how long it takes to run one kilometer. Both describe the same thing from different directions:

- Speed = distance ÷ time
- Pace = time ÷ distance

Among runners, **pace (min/km)** is more commonly used as a metric because it describes performance more intuitively in training.

## Tips for Improving Speed

1. **Interval training** — varying speed sections develop maximum oxygen uptake (VO₂max)
2. **Long run** — weekly longer run improves endurance
3. **Steady pace** — start moderately, don't sprint right at the beginning
4. **Rest** — development happens during recovery, not during exercise
5. **Muscle care** — regular stretching and strengthening exercises prevent injuries

## Running Speed Calculation Formula

The calculator uses simple basic formulas:

**Speed** = distance (km) ÷ time (h)

**Pace** = time (min) ÷ distance (km)

**Estimated time** = pace × race distance (km)

Note: estimated times assume a steady pace throughout the distance. On longer distances, actual time is usually slightly longer due to fatigue.
