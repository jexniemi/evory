## What Is a Random Color Generator?

A **random color generator** picks colors using a pseudo-random number generator (PRNG) to produce values across the full 24-bit color space — roughly 16.7 million possible colors. Each color is simultaneously expressed in three industry-standard formats: **HEX**, **RGB**, and **HSL**, so you can copy whichever format your project needs.

## Color Format Explained

### HEX (Hexadecimal)

The most common format for web design. A six-character string prefixed with `#`, where each pair of characters represents the red, green, and blue channels in base-16 (0–255 → 00–FF).

> **Formula:** `#RRGGBB` — e.g., `#3A7BD5`

### RGB (Red, Green, Blue)

Each channel is a decimal integer from 0 to 255. Used extensively in CSS and image processing.

> **Formula:** `rgb(R, G, B)` — e.g., `rgb(58, 123, 213)`

### HSL (Hue, Saturation, Lightness)

A more human-intuitive model. **Hue** is a degree on the color wheel (0–360°), **Saturation** is the intensity (0–100%), and **Lightness** is the brightness (0–100%).

> **Formula:** `hsl(H, S%, L%)` — e.g., `hsl(214, 63%, 53%)`

### Conversion: RGB to HSL

Given R, G, B normalized to the range 0–1:

> **L** = (max + min) / 2
>
> **S** = (max − min) / (2 − max − min) when L > 0.5, otherwise (max − min) / (max + min)

The Hue is calculated based on which channel (R, G, or B) is the maximum.

## Color Theory Basics

Understanding color relationships helps you build harmonious palettes:

- **Complementary:** Two colors opposite on the color wheel (180° apart). High contrast, vibrant.
- **Analogous:** Three colors adjacent on the wheel (30° apart). Harmonious and calm.
- **Triadic:** Three colors evenly spaced (120° apart). Balanced yet varied.
- **Monochromatic:** Same hue at different saturations and lightnesses. Elegant and cohesive.

## Accessibility and Contrast

When using colors in UI design, ensure sufficient contrast between text and background. The WCAG 2.1 standard requires a contrast ratio of at least **4.5:1** for normal text and **3:1** for large text. Tools like the [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) help verify compliance.
